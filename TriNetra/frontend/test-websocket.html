üîß Troubleshooting Steps

  1. Restart Vite server: Stop (Ctrl+C) and run npm run dev again
  2. Try different port: Use vite --port 3000
  3. Disable HMR: Try npm run dev:no-hmr
  4. Check firewall: Ensure port 5175 is not blocked
  5. Try different browser: Test in Chrome or other browsers
  6. Use localhost: Try http://127.0.0.1:5175 instead
